Revisão: Básico do CSS & HTML5

Seção 1 - 6

HTML é a linguagem utilizada para estruturar páginas web. Ele organiza o conteúdo exibido no navegador, permitindo a inserção de texto, imagens, links e outros elementos visuais.

Algumas das suas caracteristicas são:

  1. Estrutura do HTML

    2. Os elementos HTML são compostos por uma tag de abertura, um conteúdo e uma tag de fechamento. Alguns elementos, chamados de "void", não possuem fechamento. Os atributos adicionam informações extras aos elementos, como definir o destino de um link ou configurar o tamanho de uma imagem.

  2. Hierarquia e Organização

    3. O HTML permite que elementos sejam aninhados dentro de outros, criando uma estrutura organizada. Existem dois tipos principais de elementos: os que ocupam toda a largura disponível na página e os que se ajustam dentro de um trecho específico de texto.

  3. Estrutura Básica do HTML

    4. Todo documento HTML deve conter uma estrutura essencial que define a página, incluindo o cabeçalho com metadados, o título e o corpo, onde o conteúdo principal é inserido.

  4. Elementos Comuns do HTML
      Os principais elementos incluem:
    • Títulos e subtítulos para organizar o conteúdo;
    • Parágrafos para blocos de texto;
    • Links para navegação entre páginas;
    • Imagens para exibição de conteúdos visuais;
    • Metadados que ajudam na indexação da página em mecanismos de busca.
  5. Atributos, Elementos e Separadores

    6. Os elementos são os blocos fundamentais de uma página web, responsáveis por exibir diferentes tipos de conteúdo. Eles podem conter atributos, que fornecem informações adicionais e personalizam o comportamento dos elementos, como identificação, estilos e links.

    Já os separadores são utilizados para organizar e estruturar melhor o conteúdo da página. Entre os mais comuns, estão a quebra de linha, que separa textos em diferentes linhas, o espaçamento entre elementos, ajustado por margens e preenchimentos, e os divisores de seção, que agrupam conteúdos relacionados para melhor organização.
